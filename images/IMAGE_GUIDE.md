# 몬스터 헌터 게임 이미지 가이드라인

## 폴더 구조
```
images/
├── monsters/      # 몬스터 이미지 (기본 + 진화)
├── effects/       # 공격/스킬 이펙트 이미지
└── backgrounds/   # 탐험 지역 배경 이미지
```

---

## 1. 몬스터 이미지 (monsters/)

### 이미지 규격
- **크기**: 256x256 픽셀 (정사각형)
- **비율**: 1:1
- **형식**: PNG (투명 배경 권장)
- **스타일**: 귀여운 일러스트 또는 픽셀아트 스타일, 포켓몬/디지몬 느낌

### 파일 명명 규칙
- 기본: `{몬스터ID}.png`
- 진화: `{몬스터ID}_evo.png`

## 2. 공격/스킬 이펙트 이미지 (effects/)

### 이미지 규격
- **크기**: 128x128 픽셀 (정사각형)
- **비율**: 1:1
- **형식**: PNG (투명 배경 필수)
- **스타일**: 에너지/마법 이펙트, 애니메이션 프레임용
